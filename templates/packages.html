项目结构

项目概述

FeedForge是一个功能强大的RSS生成器，能够从不包含RSS的网站创建RSS订阅源。主要特点包括：

  • 从任何网站提取内容并生成标准RSS格式
  • 支持多种选择器策略找到正确的内容
  • 集成通知系统，支持多种通知方式
  • 用户友好的Web界面
  • 可视化选择器帮助工具

核心组件

核心文件

文件名 描述
app.py Flask应用主文件，处理所有Web路由和请求
rss_generator.py RSS生成核心逻辑，处理网页内容提取和RSS文件生成
notification_system.py 通知系统核心，处理各种通知方式的基础功能
notification_integration.py 通知系统集成模块，实现各种具体通知方式
logging_config.py 日志系统配置文件
test_selector.py 选择器测试工具，用于测试和调试CSS选择器

目录结构

目录名 描述
templates/ HTML模板文件目录，包含所有Web界面模板
static/ 静态资源目录，包含CSS、JavaScript、图片等
logs/ 日志文件目录，存储应用运行日志
rss_files/ 生成的RSS文件存储目录
rss_output/ RSS输出临时目录

依赖包

包名 版本 用途
Flask 2.0.1 Web应用框架，提供路由和请求处理
APScheduler 3.9.1 任务调度器，用于定时更新RSS
requests 2.28.1 HTTP请求库，用于获取网页内容
beautifulsoup4 4.11.1 HTML解析库，用于提取网页内容
feedgen 0.9.0 RSS/Atom feed生成库
selenium 4.8.0 浏览器自动化工具，用于处理动态加载内容
lxml 4.9.2 XML/HTML处理库，提供高效解析
python-dateutil 2.8.2 日期时间处理工具

功能模块

负责从网页提取内容并生成RSS feed：

  • 支持多种选择器策略 (CSS, XPath)
  • 处理不同类型的网页结构
  • 支持动态加载内容的网页
  • 生成符合标准的RSS文件

主要文件：rss_generator.py

提供用户友好的Web界面：

  • 添加和管理RSS订阅
  • 查看生成的文章
  • 配置通知设置
  • 可视化选择器帮助工具

主要文件：app.py，templates/目录

管理多种通知方式：

  • 邮件通知
  • Webhook通知
  • 桌面通知
  • Telegram通知
  • 企业微信通知

主要文件：notification_system.py，notification_integration.py

帮助用户找到正确的选择器：

  • 可视化选择网页元素
  • 自动生成CSS选择器
  • 测试选择器有效性

主要文件：static/selector_helper.html，test_selector.py

部署选项

本地部署

  1. 克隆仓库
  2. 安装依赖：pip install -r requirements.txt
  3. 运行应用：python app.py
  4. 访问：http://localhost:5000

Docker部署

  1. 使用提供的Dockerfile构建镜像
  2. 或使用docker-compose：docker-compose up -d
  3. 访问：http://localhost:5000
